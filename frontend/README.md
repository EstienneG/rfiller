# Rfiller

## Overview

This is a Vue.js application. Follow the steps below to install dependencies, build, and run the project locally using [pnpm](https://pnpm.io/).

---

## Prerequisites

- **Node.js**: Make sure you have Node.js (v14 or later) installed. You can check your version with:
  ```bash
  node -v
  ```

- This project uses pnpm for package management. If you donâ€™t have pnpm installed, you can install it via npm:
  ```bash
  npm install -g pnpm
  ```

- Install dependencies:
  ```bash
  pnpm install
  ```

## Run frontend application :

- To start a local development server, run:
  ```bash
  pnpm run dev
  ```

- To build the application for production, use:
  ```bash
  pnpm run build
  ```

- You can preview the production build locally using:
  ```bash
  pnpm run preview
  ```
